## Instruções de Execução do Projeto

### **AMBIENTE DE DESENVOLVIMENTO**

Primeiro, clone este reposiótio:

```bash
$ git clone git@github.com:lkasvr/ubots.git
```
Depois, execute os comandos abaixo na raíz do projeto:

```bash
npm install
```

```bash
npm run dev
```
